Ejercicio 1: Números del 1 al 10

Escribe un programa que use un bucle while para imprimir los números del 1 al 10.

Ejercicio 2: Suma de números positivos

Crea un programa que permita al usuario introducir números positivos. El programa debe sumar esos números y terminar cuando el usuario introduzca un número negativo.

Ejercicio 3: Adivina el número

Escribe un programa donde el usuario trate de adivinar un número entre 1 y 100. El programa debe seguir pidiendo números hasta que el usuario adivine el número correcto. Proporciona pistas como “más alto” o “más bajo”.

Ejercicio 4: Tabla de multiplicar

Pide al usuario un número y usa un bucle while para imprimir la tabla de multiplicar de ese número hasta el 10.

Ejercicio 5: Cuenta regresiva

Crea un programa que haga una cuenta regresiva desde un número introducido por el usuario hasta 0.

Ejercicio 6: Suma de dígitos

Pide al usuario un número y usa un bucle while para calcular la suma de sus dígitos.

Ejercicio 7: Menú interactivo

Crea un menú que siga mostrando opciones hasta que el usuario elija “Salir”. Por ejemplo:
	1.	Saludar
	2.	Mostrar hora actual
	3.	Salir

Ejercicio 8: Número mayor

Pide al usuario que introduzca números hasta que escriba “0”. Al final, muestra cuál fue el número más grande que introdujo.

Ejercicio 9: Número primo

Pide al usuario un número y utiliza un bucle while para determinar si es primo.

Ejercicio 10: Secuencia de Fibonacci

Genera los primeros n números de la secuencia de Fibonacci, donde n lo ingresa el usuario.
